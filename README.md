## 1.작품소개
> * RC카의 1인칭 시점을 VR로 보며 조종함으로써 실제 주행을 체험하는 것 같은 재미 제공

> * Score Mode를 통하여 트랙 내에서 주행 할 시, 주행 중 인식한 정보를 증강현실 객
체화 하며 그 정보를 토대로 주행점수를 평가

> * Normal Mode는 영상 처리가 없는 실시간 영상만을 제공하며, 트랙 외에서도 자유롭
게 주행이 가능
